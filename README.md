# 股息追踪 Portfolio

本地使用的股票持仓管理工具，支持 A 股、港股、美股。

## 功能

- **持仓管理** — 添加、编辑、删除股票持仓
- **交易记录** — 记录每次买入/卖出明细，自动计算加权均价和总投入
- **实时行情** — 自动获取最新股价（数据来源：腾讯行情接口）
- **盈亏计算** — 基于最新价计算持仓市值、当日盈亏和总盈亏
- **股息分析** — 显示每股分红、股息率、成本息率、预计年分红
- **账户余额** — 可手动设置的余额，买入/卖出时自动更新
- **行业图标** — 根据股票名称自动匹配行业图标
- **智能识别** — 输入代码自动识别市场并填充名称
- **数据持久化** — 数据存储在浏览器 localStorage，关闭页面不丢失

## 使用

直接打开 `收息日记.html` 即可使用。

实时行情功能在 `file://` 协议下可能受限，如果遇到行情刷不出来，运行：

```bash
python3 -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 数据存储

所有数据存储在浏览器本地存储（localStorage）中，不会上传到任何服务器。
